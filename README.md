# CIS228 Hello Expo - Kuromi Interactive App

A cute and interactive React Native app featuring Kuromi! This app allows users to enter their name and receive a personalized greeting with an adorable Kuromi profile picture.

# Features

- 🖤 Custom Kuromi profile picture
- 💀 Personalized greeting that updates in real-time
- ✨ Text input for user's name
- 🎀 Pink "SAY HELLO" button that triggers an alert

# Installation & Setup


## 1. Clone the repository

bash

git clone https://github.com/YOUR-USERNAME/cis228-hello-expo.git

## 2. Navigate to the project folder

bash

cd cis228-hello-expo

## 3. Install dependencies

bash

npm install

**This will install all required packages including React Native, Expo, and other dependencies listed in package.json.**

# Running the App
## Method 1: Run on Web Browser
bash

npm run web

**This will open the app in your default web browser at http://localhost:8081**

## Method 2: Run on Physical Device (iOS/Android)

### 1. Start the Expo development server

bash

npm start

**This will start the Metro bundler and show a QR code in the terminal.**

### 2. Open on your phone

- Install the Expo Go app from the App Store (iOS) or Google Play Store (Android)

- Scan the QR code shown in the terminal with your phone's camera (iOS) or the Expo Go app (Android)

- The app will load on your device

## Method 3: Run on Emulator/Simulator
### For Android Emulator:

bash

npm run android

### For iOS Simulator (Mac only):

bash

npm run ios

# How to Use the App
- Wait for the app to load completely

- You'll see a Kuromi profile picture and a greeting

- Tap the text input field and type your name

- Press the pink "SAY HELLO" button

- A popup alert will appear with a personalized greeting: "Nice to meet you, [your name]! 😊"
